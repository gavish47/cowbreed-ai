How It Works
01
📸

Upload a clear cattle photo via mobile or desktop. Side-profile images yield best results for morphological feature extraction.
02
⚡
AI Inference
ResNet-50 processes the image through convolutional layers — extracting coat texture, horn shape, body structure and colour patterns automatically.
03
📊
Breed + Health Insights
Get the top predicted breed with confidence, origin, description, ranked alternatives, and a curated list of breed-specific disease risks.
Technology Stack
🧠
TensorFlow + ResNet-50
Transfer learning from ImageNet weights fine-tuned on 14,000+ Indian cattle images. Data augmentation (rotation, flipping, scaling) reduces overfitting.
🐍
Python + Flask Backend
Flask REST API handles image uploads, PIL preprocessing (resize, normalize), model inference, and structured JSON prediction responses.
📈
Grad-CAM Explainability
Gradient-weighted Class Activation Maps highlight which image regions drove the breed prediction — improving transparency for non-expert users.
🌐
Web + Mobile Deployment
Lightweight frontend with drag-and-drop upload, animated confidence bars, breed disease insights, and downloadable reports accessible from any device.
Model Performance
Training vs Validation Loss
Training loss decreases from ~1.15 to below 0.15 across epochs, showing strong feature convergence. Validation loss stabilises well with regularisation applied via augmentation and dropout layers.
Classification Results
Achieves 85%+ validation accuracy across 8 breeds. Strong performance on Lakhimi, Umblachery, and Siri. Sahiwal occasionally misclassified as Lakhimi due to similar coat patterns — a known challenge in fine-grained breed recognition.

Why CowBreed AI?
The Problem We Solve
India is home to over 50 indigenous cattle breeds — yet most go unidentified, misclassified or ignored. Manual identification requires trained experts who are scarce in rural areas. CowBreed AI bridges this gap.
